NAME

pstree − display a tree of processes

SYNOPSIS

pstree [--all] [-a | --arguments] [-C | --color attr] [--colorize] [-s | --contains pattern] [--cpu] [-U | --exclude-root] [-h | --help] [-i | --ibm-850] [-l | --level level] [-m | --memory] [-g | --pgid] [--pid pid] [--rainbow] [-p | --show-pids] [-t | --threads] [--user user] [-u | --utf-8] [-v | --vt-100-8] [-V | --version-8] [-w | --wide]

DESCRIPTION

pstree pstree shows running processes as a tree. The tree is rooted at launchd. If --pid is specified, the tree will be rooted at the specified PID. If --user is specified, all process trees rooted at processes owned by that user are shown.

OPTIONS

−-all

Equivalent to -a -c -g -m -p -t

−a, −-arguments

Show command line arguments after the process name.

−C, −-color attribute

Color the process name by given attribute. Currently pstree accepts the values age, cpu, and me. With age, processes newer than 60 seconds are green, newer than an hour are yellow, and the remaining are red. With cpu, processes using less than 5% CPU are green, greater than 5% and less than 15% are yellow, and the remaining are red. With mem, processes using less than 10% memory are green, greater than 10% and less than 20% are yellow, and the remaining are red. This option cannot be used with either --colorize or --rainbow.

--colorize

Colorize the pstree output. This option is only visible in the help if your terminal supports color. This option is not available if your terminal doesn’t support at least 8 color output. This option cannot be used with either --color or --rainbow.

−s, −-contains pattern

Show only branches containing processes with pattern in the command line.

−c, −-cpu

Show the CPU utilization for each process in the list using the format (c: 0.00%).

−U, −-exclude-root

Don’t show branches containing only root processes. This option cannot be used with --user.

−h, −-help

Display a help message and exit.

−i, −-ibm-850

Use IBM-850 line drawing characters.

−l, −-level level

Traverse the tree level levels deep.

−m, −-memory

Show the memory usage for each process in the list using the format (m: 0.00 MiB).

−g, −-pgid

Show PGIDs. Process Group IDs are shown as decimal numbers in parentheses after each process name. If this option is selected, the PGID will be displayed after the PID.

−-pid PID

Show only branches containing process PID.

--rainbow

Colorize the Display the output in a really annoying rainbow pattern. You’ve been warned. This option is not available if your terminal doesn’t support 256 color output. This option cannot be used with either --color or --colorize.

−p, −-pids

Show PIDs. Process IDs are shown as decimal numbers in parentheses after each process name.

−t, −-threads

Show the number of threads for each process in the list using the format (t: xx).

−-user user

If this option is used, all process trees rooted at processes owned by user are shown. This option cannot be used with --exclude-root.

−u, −-utf-8

Use UTF-8 (Unicode) line drawing characters.

−v, −-vt-100

Use VT-100 (Unicode) line drawing characters.

−V, −-version

Display version information.

−w, −-wide

Display long lines. By default, lines are truncated to either the COLUMNS environment variable or the display width. If neither of these methods work, the default of 132 columns is used.

EXAMPLES

To display help:

pstree --help

To show processes owner by bob and also display the command line arguments:

pstree --user bob -a

To show the process ID, process group ID, command line arguments, use a wide format, and use UTF-8 graphics:

pstree -p -g -a -w -u
